What is QctoupsQL?

Qt + Octoups + PostgreSQL = QctoupsQL.

PostgreSQL the client for handle of base Exim written on QT.

  • Add ,view ,delete - Users,Domains,Aliases,UserForward.
  • Find Users,Aliases and UserFoward
  • Set "Quote".
  • Set individual spam score and action for a spam.

What are the Requirements?

  • Qt>=4.2
  • libpq>=8.0
  • Exim>=4.3x
